## Overview

Global Diary is a social media platform with real-time feeds and
interactions.

I worked across the whole thing: a Node.js backend on PostgreSQL and Redis
with Socket.IO powering the real-time layer, a React/Next.js + Tailwind
frontend, and the DevOps side — provisioning and running the platform on
AWS.

## Working at scale

Global Diary was my education in scale. A social platform lives or dies on
how it behaves with lots of concurrent users hitting real-time feeds, so much
of the project was optimization: tuning PostgreSQL queries, leaning on Redis
where it earned its keep, and shaping the Socket.IO layer so real-time stayed
real under load. Those lessons have paid off in every project since.
