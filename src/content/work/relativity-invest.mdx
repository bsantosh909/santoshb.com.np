## Overview

Relativity Invest is an AI-driven investment platform built around a simple
thesis: **the best teams win — find them.**

I work across the entire product: backend services in Node.js and Python with
Kafka moving data through the pipeline, a React/Next.js + Tailwind web
frontend, the React Native mobile app, and the AWS infrastructure that runs
it all — EC2, ECR, ECS, RDS, Route 53 and more.

## Data at scale

At its core this is a data problem: huge volumes flowing in that have to be
normalized, filtered, and cleaned before the AI can say anything useful about
a team. That pipeline — Kafka moving the data, Python and Node.js services
shaping it — is where I've learned the most about working with data at
serious scale. The other half of the job is making it usable: the web
frontend, the services around it, and the React Native app that puts it all
in investors' hands.
